# ProgramacionVet
Proyecto de sistema de Veterinaria para la materia "Programación de Software" 2025-2
## Descripción del Proyecto

Sistema integral de gestión para clínica veterinaria que permite administrar:
-  **Mascotas y clientes**
-  **Servicios veterinarios** 
-  **Productos e inventario**
-  **Transacciones y métodos de pago**
-  **Empleados y asignaciones**

## Funcionalidades Principales
Gestión de Clientes y Mascotas
- Registrar y consultar clientes
- Gestionar mascotas y sus dueños
- Historial de mascotas por cliente

Servicios Veterinarios
- Catálogo de servicios por categoría
- Asignación de tiempo estimado
- Registro de observaciones médicas

Transacciones y Pagos
- Procesar ventas de productos
- Registrar servicios realizados
- Múltiples métodos de pago con comisiones

Gestión de Empleados
- Registro de personal
- Asignación de servicios a empleados
- Control por cargos y responsabilidades
  
##  Objetivos

- Digitalizar los procesos de una clínica veterinaria
- Gestionar eficientemente clientes, mascotas y historiales médicos
- Controlar inventario de productos y servicios
- Generar reportes y transacciones financieras
